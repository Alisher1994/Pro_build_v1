# Как импортировать CSV файлы с данными сметы

## Инструкция по импорту

1. **Откройте смету** в системе (нажмите на нужную смету в списке)

2. **Нажмите кнопку "Импорт XML"** в верхней панели окна сметы (справа вверху)

3. **Выберите тип документа:**
   - Для файла `example_import_type1_utf8.csv` выберите **"Тип 1: Упрощенный"**
   - Формат: № | Тип ресурса | Название | Ед.изм | Кол-во | Цена

4. **Загрузите файл:**
   - Перетащите CSV файл в область загрузки
   - Или нажмите на "выберите файл" и выберите файл

5. **Проверьте данные** на шаге "Форматирование"

6. **Выполните импорт** на последнем шаге

## Формат файла

Ваш файл `example_import_type1_utf8.csv` соответствует формату **Тип 1**:
- Колонки: № | Тип ресурса | Название | Ед.изм | Кол-во | Цена
- **Этапы** определяются по строкам без номера (например: "ЭТАП 1: ЗЕМЛЯНЫЕ РАБОТЫ")
- **Виды работ** - строки с целым числом (1, 2, 3...)
- **Ресурсы** - строки с номером через точку (1.1, 1.2, 2.1...)

## Поддерживаемые форматы

- CSV (с разделителем табуляции или запятой)
- Excel (XLS, XLSX)
