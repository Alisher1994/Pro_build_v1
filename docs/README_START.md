# Инструкция по запуску ProBIM

Данный проект состоит из трех основных частей: **Frontend** (интерфейс), **Backend** (API и логика) и **Ollama AI** (интеллектуальные функции).

## Быстрый запуск (Рекомендуется)

Используйте файл автоматического запуска в корне проекта:
1.  Запустите **`start-all.bat`**.
    - Скрипт проверит, запущен ли **Ollama**.
    - Закроет старые процессы, если они зависли.
    - Запустит Backend в режиме разработки (`dev`).
    - Запустит Frontend через Python сервер.

## Настройка AI (Ollama)

Для работы функций генерации инструкций и нормативов необходимо, чтобы были скачаны соответствующие модели.
Выполните в консоли:
```bash
ollama pull llama3.2:1b
ollama pull llama3.2:3b
```

## Дополнительные функции

### 1. Парсинг рейтинга mc.uz
Функция автоматического получения рейтинга субподрядчика. 
- Работает через Puppeteer в бэкенде.
- Для обновления рейтинга используйте кнопку (иконка стрелок) в карточке субподрядчика на вкладке тендеров.

### 2. Расчет нормативов и генерация инструкций
- Инструкции используют модель **`llama3.2:1b`** для скорости.
- Нормативы используют модель **`llama3.2:3b`** для точности.
- Настройки моделей находятся в файле `backend/.env`.
