# Руководство по установке окружения (после переустановки Windows)

Этот файл содержит список необходимых программ и шагов для развертывания проекта **Pro_build** на чистой системе.

## 1. Необходимые программы

Перед началом работы установите следующее ПО:

1.  **Node.js (LTS версия)**
    *   **Зачем:** Для запуска серверной части (Backend) и инструментов сборки.
    *   **Где скачать:** [nodejs.org](https://nodejs.org/) (выбирайте версию LTS, например 20.x или 22.x).
    *   *Проверка:* В терминале введите `node -v` и `npm -v`.

2.  **Git**
    *   **Зачем:** Для управления версиями кода и скачивания репозитория.
    *   **Где скачать:** [git-scm.com](https://git-scm.com/downloads).
    *   *Настройка:* После установки введите в терминале:
        ```bash
        git config --global user.name "Ваше Имя"
        git config --global user.email "ваш@email.com"
        ```

3.  **Visual Studio Code (VS Code)**
    *   **Зачем:** Редактор кода.
    *   **Где скачать:** [code.visualstudio.com](https://code.visualstudio.com/).

## 2. Рекомендуемые расширения для VS Code

После установки VS Code рекомендуется установить эти расширения (Ctrl+Shift+X):

*   **Prisma** (Prisma.prisma) — для подсветки синтаксиса схем баз данных.
*   **ESLint** (dbaeumer.vscode-eslint) — для проверки ошибок в JS/TS коде.
*   **Prettier** (esbenp.prettier-vscode) — для форматирования кода.
*   **GitHub Copilot** — если вы используете ИИ-помощника.

## 3. База данных

В проекте используется **SQLite**.
*   **Установка не требуется:** SQLite встроен в библиотеку Prisma, ничего отдельно скачивать и устанавливать не нужно. Файл базы данных (`dev.db`) создастся автоматически внутри папки `backend/prisma`.

## 4. Пошаговая инструкция по запуску проекта

После того как вы скопировали папку с проектом обратно на компьютер:

1.  **Откройте проект в VS Code:**
    *   Нажмите `File` -> `Open Folder` и выберите корневую папку `Pro_build`.

2.  **Установите зависимости (библиотеки):**
    Откройте терминал в VS Code (Ctrl+`) и выполните команды по очереди:

    *   **Корневые зависимости:**
        ```bash
        npm install
        ```
    *   **Зависимости Backend:**
        ```bash
        cd backend
        npm install
        ```
    *   **Зависимости Frontend:**
        ```bash
        cd ../frontend
        npm install
        ```

3.  **Инициализация Базы Данных:**
    Находясь в папке `backend` (в терминале), выполните:
    ```bash
    npx prisma generate
    npx prisma db push
    ```
    *Это создаст файл базы данных `dev.db` на основе вашей схемы.*

4.  **Запуск проекта:**
    Вернитесь в корень проекта и запустите:
    ```bash
    cd ..
    npm run dev
    ```
    *Эта команда запустит и Backend (порт 3001), и Frontend (порт 8000).*

## 5. Полезные команды (Шпаргалка)

*   `npx prisma studio` (в папке backend) — открывает визуальный редактор базы данных в браузере.
*   `npm run dev` (в корне) — запуск всего проекта.
